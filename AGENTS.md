# AGENTS.md

## 목적
이 저장소는 일관된 코드 품질과 유지보수를 위해 다음 규칙을 따른다.

## 코드 스타일
- 함수는 단일 책임 원칙(SRP)을 따른다
- 1 함수는 최대 50줄 이하
- 변수명은 의미 기반으로 작성 (축약 금지)

## 아키텍처
- UI / Service / Data Layer 분리
- 비즈니스 로직은 반드시 Service Layer에 위치

## 금지 사항
- 하드코딩 금지
- console.log 남기지 말 것
- 중복 코드 작성 금지

## 테스트
- 모든 핵심 로직은 unit test 포함
- edge case 반드시 고려

## PR 규칙
- 작은 단위로 PR 생성
- 기능 단위로 분리

## AI 행동 규칙
- 기존 코드 스타일을 반드시 따를 것
- 새로운 패턴 도입 시 이유 명시
